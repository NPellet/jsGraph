setStyle(allStyles, modifiers, selectionMode)

Sets style to the scatter points First argument is the style applied by default to all points Second argument is an array of modifiers that allows customization of any point of the scatter plot. Data for each elements of the array will augment allStyles, so be sure to reset the style if needed. All parameters - except shape - will be set as parameters to the DOM element of the shape

Parameters:
Name Type Argument Default Description
allStyles Object

The general style for all markers
modifiers Object <optional>

The general style for all markers
selectionMode String <optional>
 "unselected"

The selection mode to which this style corresponds. Default is unselected
Example
var modifiers = [];
modifiers[ 20 ] = { shape: 'circle', r: 12, fill: 'rgba(0, 100, 255, 0.3)', stroke: 'rgb(0, 150, 255)' };
serie.setStyle( { shape: 'circle', r: 2, fill: 'rgba(255, 0, 0, 0.3)', stroke: 'rgb(255, 100, 0)' }, modifiers ); // Will modify scatter point nÂ°20
