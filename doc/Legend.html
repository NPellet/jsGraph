Methods

setPosition(position, alignToX, alignToY)

Sets the position of the legend

Parameters:
Name Type Description
position Position

the position to set the legend to versus the graph main axes (Graph#getXAxis and Graph#getYAxis)
alignToX String

"right" or "left". References the legend right or left boundary using the position parameter
alignToY String

"top" or "bottom". References the legend top or bottom boundary using the position parameter
Example
legend.setPosition( { x: 'max', y: '0px' }, 'right', 'top' ); // The rightmost side of the legend will at the maximum value of the axis, and will be positioned at the top
